Задание 2-1: Многопоточность
Требования:
•Используйте библиотеку pthread
•Компиляция: gcc -pthread -o task task.c
•Функции: pthread_create(), pthread_join(), pthread_exit()

Вариант 6: Создать 4 потока. Каждый вычисляет факториал числа (5,6,7,8). Результаты
в глобальном массиве. Главный поток выводит все

Используемые функции:
pthread_create() - создание нового потока
pthread_join() - ожидание завершения потока
pthread_exit() - завершение потока
Тип pthread_t - идентификатор потока

Все 4 потока создаются почти одновременно
Потоки выполняются параллельно
Порядок завершения потоков непредсказуем
Главный поток блокируется на pthread_join() до завершения всех дочерних потоков

Для запуска используется ./task

Пример вывода:
4 потока:
Поток 0: вычислил 5! = 120
Поток 1: вычислил 6! = 720
Поток 2: вычислил 7! = 5040
Поток 3: вычислил 8! = 40320

Результаты:
Факториал 5! = 120
Факториал 6! = 720
Факториал 7! = 5040
Факториал 8! = 40320
