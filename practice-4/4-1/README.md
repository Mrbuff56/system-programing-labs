Требования:

• Проверяйте результат выделения памяти.
• Компиляция: gcc -Wall -Wextra -o task task.c. 

Вариант 6. Реализовать динамический список строк, вводимых пользователем, 
затем вывести их в обратном порядке.

Используемые функции:
malloc() - выделяет блок памяти заданного размера в куче (heap)
realloc() - изменяет размер ранее выделенного блока памяти
free() - освобождает ранее выделенную память
fgets() - читает строку из стандартного ввода
strlen() - возвращает длину строки
strcpy() - копирует строку из одного места в другое

Алгоритм работы программы:
Основной цикл ввода в бесконечном цикле:
Выводит приглашение к вводу (printf)
Читает строку через fgets() в буфер
Удаляет символ перевода строки (\n) из буфера
Проверяет условие завершения: если строка пустая (длина 0) - выход из цикла
Вычисляет размер строки: (len + 1) * sizeof(char)
Вызывает malloc() для выделения памяти под новую строку
Проверяет результат: если NULL - выводит ошибку и освобождает ранее выделенную память
Копирует содержимое буфера в выделенную область с помощью strcpy()
Вызывает realloc() для увеличения размера массива lines на один элемент
Проверяет результат: если NULL - обрабатывает ошибку
Сохраняет указатель на новую строку в массиве
Увеличивает счетчик count

После завершения ввода:
Выводит заголовок "Строки в обратном порядке:"
В цикле от count - 1 до 0 Выводит строку с номером
В цикле от 0 до count - 1 вызывает free() для каждой строки
Вызывает free() для массива указателей lines
