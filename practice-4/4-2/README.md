Задание 4-2: Отображение файлов в память mmap()
Требования:
• Используйте open(), fstat(), mmap(), munmap(), close().
• Обрабатывайте ошибки.
• Компиляция: gcc -Wall -Wextra -o task task.c.

Вариант 6. Отобразить двоичный файл с массивом целых чисел и вычислить среднее значение.

Используемые функции:
open() - открывает файл
close() - закрывает файл
fstat() - получает информацию о файле по его дескриптору
mmap() - отображает файл в память процесса
munmap() - удаляет отображение файла из памяти процесса

Алгоритм работы программы:
Проверяет аргументы командной строки
Открывает файл
Получение информации о файле
Проверка корректности файла
Вычисление количества элементов
Отображает весь файл в память mmap()
Закрытие файлового дескриптора close(fd) 
Вывод и суммирование чисел
Вычисление среднего значения
Вывод результатов количество элементов и среднее значение 
Освобождает память, занятую отображением munmap()
