Задание 3-1: Неименованные каналы pipe()
Требования:
• Используйте pipe(), fork(), read(), write(), close().
• Не используйте высокоуровневый ввод-вывод fopen(), fprintf(), fscanf().
• Обрабатывайте ошибки через perror().
• Компиляция: gcc -Wall -Wextra -o task task.c.

Вариант 6. Родитель читает строку с клавиатуры и передаёт её дочернему через pipe(). 
Дочерний переводит все строчные буквы в заглавные и выводит результат.

Используемые системные вызовы:
pipe() - создает неименованный канал (pipe), возвращает два файловых дескриптора: pipefd[0] (чтение) и pipefd[1] (запись)
fork() - создает дочерний процесс
read() - чтение данныx
write() - запись данных
close() - закрывает файловый дескриптор
wait() - ожидает завершения дочернего процесса

Алгоритм работы:
Родительский процесс:
Читает строку с клавиатуры через fgets()
Записывает строку в pipe через write()
Закрывает свой конец pipe
Ожидает завершения дочернего процесса через wait()

Дочерний процесс:
Читает строку из pipe через read()
Преобразует все строчные буквы в заглавные с помощью islower() и toupper()
Выводит результат на экран
Закрывает свой конец pipe
