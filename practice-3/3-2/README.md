Задание 3-2: Именованные каналы FIFO
Требования:
• Используйте mkfifo(), open(), read(), write(), close(), unlink().
• Обрабатывайте ошибки.
• Компиляция: gcc -Wall -Wextra -o task task.c.

Вариант 6. Один процесс каждую секунду отправляет текущее время в FIFO,
второй принимает и записывает в time.log.

Используемые функции:
mkfifo() - создает именованный канал (FIFO) в файловой системе
open() - открывает для чтения или записи
read() - чтение данныx
write()	- запись данных
close()	- закрывает файловый дескриптор
unlink() - удаляет FIFO из файловой системы
fork()	- создает дочерний процесс
wait()	- ожидает завершения дочернего процесса
time()/localtime()/strftime()	- получает и форматирует время

Алгоритм работы программы:
Инициализация main():
Удаляет FIFO если он существует unlink()
Создает именованный канал mkfifo()
Создает дочерний процесс fork()

Дочерний процесс:
Открывает FIFO для чтения open()
Открывает/создает лог-файл time.log для записи
В бесконечном цикле читает данные из FIFO
Удаляет символ новой строки из полученных данных
Записывает время в лог-файл
При ошибке или закрытии FIFO завершает работу

Родительский процесс:
Ждет 1 секунду (дает получателю время на открытие FIFO)
Открывает FIFO для записи open()
В цикле из 10 итераций:
Получает текущее время
Форматирует его в строку
Записывает время в FIFO
Ждет 1 секунду sleep(1)
Закрывает FIFO и завершается

Родитель ждет завершения дочернего процесса wait()
Удаляет FIFO из файловой системы unlink()
